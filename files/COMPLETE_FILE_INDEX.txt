║                    WHICH VERSION TO USE?                             ║
╚══════════════════════════════════════════════════════════════════════╝

USE NMAP VERSION IF:
────────────────────────────────────────────────────────────────────────
✓ You have nmap installed (or can install it)
✓ You can run with sudo
✓ You need professional-grade scanning
✓ You want maximum detection accuracy
✓ You're preparing for CERT-IN exam
✓ You're doing pentesting/security assessment
✓ Efficiency matters

COMMAND:
sudo python3 nmap_workflow.py -f subnets.txt


USE PYTHON VERSION IF:
────────────────────────────────────────────────────────────────────────
✓ You can't install nmap
✓ You can't run with sudo
✓ You need pure Python solution
✓ You're learning network programming
✓ You have minimal requirements
✗ Less efficient
✗ May miss some hosts

COMMAND:
python3 complete_workflow.py -f subnets.txt

NMAP VERSION (RECOMMENDED):
────────────────────────────────────────────────────────────────────────

# Install nmap first
sudo apt install nmap

# One command (scan + report)
sudo python3 nmap_workflow.py -f subnets.txt

# Scan only
sudo python3 nmap_subnet_sweep.py -f subnets.txt

# Fast scan
sudo python3 nmap_workflow.py -f subnets.txt -t T5

# Excel only
sudo python3 nmap_workflow.py -f subnets.txt -r excel


PYTHON VERSION:
────────────────────────────────────────────────────────────────────────

# No installation needed

# One command (scan + report)
python3 complete_workflow.py -f subnets.txt

# Scan only
python3 advanced_subnet_sweep.py -f subnets.txt

# Fast scan (more threads)
python3 complete_workflow.py -f subnets.txt -t 100

# Excel only
python3 complete_workflow.py -f subnets.txt -r excel


REPORT GENERATION (BOTH):
────────────────────────────────────────────────────────────────────────

# Generate report from existing scans
python3 generate_report.py -d scan_results

# Custom output name
python3 generate_report.py -d scan_results -o my_report

# CSV only
python3 generate_report.py -d scan_results -f csv
