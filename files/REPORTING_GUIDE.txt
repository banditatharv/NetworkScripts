📊 OVERVIEW
-----------
The reporting system consolidates subnet sweep results into professional
CSV/Excel reports with subnet-wise organization and detection details.


🎯 THREE WAYS TO USE
--------------------

METHOD 1: Complete Workflow (Recommended)
  → Scan subnets AND generate report in one command
  
  python3 complete_workflow.py -f subnets.txt

METHOD 2: Report Only
  → Generate report from existing scan results
  
  python3 generate_report.py -d scan_results

METHOD 3: Manual Steps
  → Step 1: Scan
      python3 advanced_subnet_sweep.py -f subnets.txt
  → Step 2: Report
      python3 generate_report.py -d scan_results


📋 REPORT FORMATS
-----------------

CSV Format:
  ✓ Compatible with all spreadsheet software

  ✓ Lightweight file size

  ✓ Professional formatting with colors
  ✓ Multiple sheets (Results + Summary)

⚡ QUICK START COMMANDS
-----------------------

# Complete workflow (scan + report)
python3 complete_workflow.py -f subnets.txt

# Fast scan with more threads
python3 complete_workflow.py -f subnets.txt -t 100

# Excel report only
python3 complete_workflow.py -f subnets.txt -r excel

# Generate report from existing scans
python3 generate_report.py -d scan_results

# Custom output name
python3 generate_report.py -d scan_results -o my_report


🔧 COMMAND OPTIONS
------------------

complete_workflow.py:
  -f, --file FILE          Subnets input file (required)
  -o, --output DIR         Output directory (default: scan_results)
  -t, --threads NUM        Scanning threads (default: 50)
  -r, --report-format FMT  csv/excel/both (default: both)

generate_report.py:
  -d, --directory DIR      Scan results directory (default: scan_results)
  -o, --output NAME        Output filename without extension
  -f, --format FMT         csv/excel/both (default: both)

🔍 REPORT ANALYSIS TIPS
------------------------

CSV Analysis:
# Count total alive IPs
grep -v "^Subnet" report.csv | awk -F',' '{sum+=$2} END {print sum}'

# Extract all IPs to file
grep -v "^Subnet" report.csv | cut -d',' -f3 | tr ',' '\n' > all_ips.txt

# Find subnets with most hosts
grep -v "^Subnet" report.csv | sort -t',' -k2 -rn | head -5

Excel Analysis:
- Use filters to find subnets with specific IP counts
- Sort by "Total Alive IPs" to prioritize scanning
- Use conditional formatting to highlight high-value subnets
- Export specific columns for tool integration


🚀 INTEGRATION WITH OTHER TOOLS
-------------------------------

Nmap Integration:
# Extract IPs from CSV
python3 -c "
import csv
with open('subnet_sweep_report.csv') as f:
    reader = csv.DictReader(f)
    for row in reader:
        ips = row['IP Addresses'].split('\n')
        with open(f\"{row['Subnet'].replace('/', '_')}_targets.txt\", 'w') as out:
            out.write('\n'.join(ips))
"

# Then scan each subnet
for f in *_targets.txt; do
    nmap -sV -iL "$f" -oA "nmap_${f%.txt}"
done

🛠️ TROUBLESHOOTING
-------------------

Issue: "openpyxl not installed"
Solution: pip install openpyxl --break-system-packages

Issue: No files found in scan_results
Solution: Run the scanner first or check directory path

Issue: Excel file won't open
Solution: Try CSV format or reinstall openpyxl

Issue: IPs not showing on separate lines in Excel
Solution: Enable "Wrap Text" in Excel cells

Issue: Report showing old data
Solution: Clear scan_results directory before new scan
